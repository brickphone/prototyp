# Mr Green Responsible Gaming Addon (Prototype)

Next.js prototype that adds a transparent responsible gaming panel with:

- Continuous risk indicator (`Låg`, `Förhöjd`, `Hög`)
- Always-visible risk dashboard with period tabs and behavioral deltas
- Automatic protective mode UI/actions when risk is high

## Run

1. `npm install`
2. `npm run dev`
3. Open `http://localhost:3000`

## Test

- `npm run test`
